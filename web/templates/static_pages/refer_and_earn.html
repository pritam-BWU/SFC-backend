How the Chain Is Recognized

1. Direct relation

A referral edge stores parent -> child when a referral code is used.

2. Full ancestry

A closure table stores ancestor -> descendant with depth for long chains.

3. Level conversion

Indirect level = depth - 1. Depth 5 means level 4 Brand Ambassador reward.
